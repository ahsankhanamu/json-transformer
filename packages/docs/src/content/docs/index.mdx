```bash
npm install @ahsankhanamu/json-transformer
```

## Quick Example

```javascript
import { evaluate } from '@ahsankhanamu/json-transformer';

const data = {
  user: { name: 'John', role: 'admin' },
  orders: [
    { product: 'Widget', price: 25.99 },
    { product: 'Gadget', price: 49.99 },
    { product: 'Gizmo', price: 15.00 }
  ]
};
```

## Export to JavaScript

Expressions compile to clean, readable JavaScript:

<CodeCompare leftLabel="Expression" rightLabel="Compiled Output">
  <Fragment slot="left">
```
orders[? .price > 20][].product | upper
```
  </Fragment>
  <Fragment slot="right">
```javascript
(input) => {
  return (input?.orders ?? [])
    .filter(i => i?.price > 20)
    .map(i => i?.product?.toUpperCase());
}
```
  </Fragment>
</CodeCompare>

Use `compile()` to get a reusable function, or `toJS()` to get the source code:

```javascript
import { compile, toJS } from '@ahsankhanamu/json-transformer';

const fn = compile('orders[? .price > 20][].product');
fn(data); // ["Widget", "Gadget"]

toJS('user.name | upper');
// "(input) => input?.user?.name?.toUpperCase()"
```
